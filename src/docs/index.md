## 📦 安装

### 使用 npm 或 yarn 安装

 推荐使用 npm 或 yarn 的方式进行开发，不仅可在开发环境轻松调试，也可放心地在生产环境打包部署使用。

#### npm

>$ npm install tw-components-library

#### yarn 

>$ yarn add tw-components-library

如果你的网络环境不佳，推荐使用 [cnpm](https://github.com/cnpm/cnpm)。
